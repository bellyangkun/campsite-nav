# 露营地活动导航网页

一套适合手机浏览的轻量级露营地图导航页面，包含顾客端与简单的后台管理。

## 功能

- **活动地点展示**：在地图上显示营地入口、活动点、服务点等标记。
- **路线指引**：顾客选择目标地点后，在地图上画出从当前位置到目的地的参考路线，并显示直线距离与目标方位。
- **实时方向**：
  - 通过 GPS `watchPosition` 实时更新顾客位置；
  - 结合 GPS heading、移动向量与设备指南针，即时显示当前行进/面朝方向。
- **后台管理**：`admin.html` 可新增、删除活动点，支持地图点击取点、数据导入/导出 JSON。

## 文件结构

```
campsite-nav/
├── index.html       # 顾客导航页
├── admin.html       # 后台管理页
├── css/style.css    # 公共样式（移动优先）
├── js/data.js       # 数据存取（localStorage）
├── js/app.js        # 导航页逻辑
├── js/admin.js      # 管理页逻辑
└── README.md
```

## 使用方式

1. 进入项目目录：
   ```bash
   cd campsite-nav
   ```
2. 启动本地服务器：
   ```bash
   python3 -m http.server 8080
   ```
3. 手机或电脑浏览器访问：
   - 导航页：`http://<你的IP>:8080/`
   - 后台管理：`http://<你的IP>:8080/admin.html`

> 提示：定位与指南针功能在 `localhost` 或 `https` 环境下效果更佳；iOS 13+ 需要用户手动授权方向传感器。

## 数据说明

- 活动点默认存储在浏览器 `localStorage` 中。
- 首次打开会自动填充示例数据。
- 可通过后台管理页导出 JSON 备份，或导入已有 JSON 覆盖当前数据。
- 如需接入后端，只需替换 `js/data.js` 中的读写实现（保持 `CampData` API 不变），其余页面无需改动。

## 提示

- 本导航使用“直线参考线”展示方向与距离，适合营地等小范围场景；若需要真实道路路径，可接入 OSRM / Mapbox Directions 等专业路径服务。
- 在后台管理页面点击地图即可自动填充经纬度，也可直接输入坐标。
